3. Access modifiers in java

public
 - if an entity is declared public - every class in the project can access this entity
default
- The code is only accessible in the same package
protected
-The code is accessible in the same package and subclasses(Child Class)
 private
- if an entity is declared private - only in the same class it is accessible
